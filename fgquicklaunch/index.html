How does this work?

Each button will open the app link for Fall Guys on the launcher that is specified (Epic Games or Steam). That link will run Fall Guys if it is installed.

Do I need to have the Epic Games/Steam Launcher open?

No, it will still work if you dont't have them open. You will just need to log into an account that has the game if you are asked to by the launcher.

Do I need to own Fall Guys on Epic Games/Steam for this to work?

Yes, you will need Fall Guys and one of the 2 launchers installed for this to work. If you have the Epic Games/Steam Launcher installed but do not own and have Fall Guys installed then it will not work.
